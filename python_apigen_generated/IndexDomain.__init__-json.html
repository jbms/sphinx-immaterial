    tensorstore_demo.IndexDomain(*, json: Any)

    Constructs an index domain from its JSON representation.

    Examples

    >>> ts.IndexDomain(
...     json={
...         "inclusive_min": ["-inf", 7, ["-inf"], [8]],
...         "exclusive_max": ["+inf", 10, ["+inf"], [17]],
...         "labels": ["x", "y", "z", ""]
...     })
{ "x": (-inf, +inf), "y": [7, 10), "z": (-inf*, +inf*), [8*, 17*) }
